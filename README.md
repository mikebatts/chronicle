# Chronicle

A daily history guessing game. One event. One image. Guess the year in 3 attempts.

**Play at:** [thischronicle.com](https://thischronicle.com)

## How to Play

1. Look at the historical event image and headline
2. Enter your guess for what year it occurred
3. You get 3 attempts — wrong guesses get clues
4. Share your result with friends

## Tech Stack

- Next.js (App Router)
- Tailwind CSS
- Static puzzle data (no backend)
- localStorage for game state

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```
